您需要开发“一个简单的火车票预订系统”，特别是，

•创建一个MongoDB数据库，保存“火车票信息”。

•为简便起见，车票只适用于在北京、上海和

广州。因此，这项服务只针对住在这三个城市的乘客。•假设数据库只存储两天的火车票信息。每张票是

由“出发地”、“目的地”、“日期”和“出发时间”指定。例如，可以设置

“2019-11-2”和“2019-11-3”作为可用票日期。

•每天，系统将保留三个时间表，以指示列车从

分别是北京、上海和广州。例如北京，它的桌子我有火车-

日程安排：上午8点（北京-上海），上午10点（北京-广州），上午11点（北京-上海）……在

每个城市的表中至少列有6列火车的时刻表。

•编写一个node.js应用程序，创建一个连接到mongobd数据库的web服务器。

•通过使用Web浏览器，用户可以打开带有URL的开始预订页面，例如http://localhost:

例如，展示“三个城市的票，…，两天，…”，然后要求

“再试一次”。

•一旦用户得到发车时刻表，他/她可以选择表中的一个列车时刻表，并且
